# capex_plan_vs_actuals.csv

**Row grain:** program + tool_id + fiscal_month + capex_category

Key columns:
- `planned_spend_usd` — baseline plan
- `actual_spend_usd` — booked/paid actual
- `forecast_spend_usd` — simple forecast model
- `install_date_planned` / `install_date_actual` — schedule signal for readiness
