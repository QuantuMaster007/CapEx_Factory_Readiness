# Data dictionary

This folder documents the synthetic/anonymized datasets used in this repo.

## Raw datasets
- `capex_plan_vs_actuals.csv` — CapEx spend by month/tool/category (plan/actual/forecast).
- `lead_times_expedite.csv` — lead time plan vs actual and expedite spend drivers.
- `facility_readiness_tasks.csv` — readiness tasks with gates, owners, dependencies, and status.

## Processed datasets
- `capex_variance_by_month_category.csv` — variance rollup for trend charts.
- `expedite_by_month_vendor.csv` — expedite trend by vendor.
- `readiness_rollup.csv` — readiness score by program/tool/site/area.
